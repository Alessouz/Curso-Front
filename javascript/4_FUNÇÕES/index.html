Oque é um função

Uma função em JavaScript é um bloco de código reutilizável projetado para realizar uma tarefa específica ou calcular um valor. As funções são uma parte fundamental da programação em JavaScript e em muitas outras linguagens de programação. Elas ajudam a organizar o código, promovem a reutilização e tornam o código mais modular.
